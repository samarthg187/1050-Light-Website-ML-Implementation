Data Format

Sensor Output

Each reading output by the sensor device is a comma-separated string of 6 values:

example_reading.csv CSV 
2024,0315,143022,450,42.9837,-81.2497
Field Format Example Description
Year yyyy 2024 4-digit calendar year
Date mmdd 0315 2-digit month + 2-digit day (March 15)
Time hhmmss 143022 24-hour hour, minute, second (14:30:22)
Luminosity lumins 450 Light level in lumens (numeric)
Latitude lat 42.9837 GPS latitude in decimal degrees
Longitude lng -81.2497 GPS longitude in decimal degrees

Key Snippets

Saving a sensor reading to localStorage:

saveReading.jsJS 
const STORAGE_KEY = 'hypercell_sensor_readings';

function saveReading(rawString) {
  // e.g. "2024,0315,143022,450,42.9837,-81.2497"
  const parts = rawString.split(',').map(p => p.trim());
  const entry = {
    yyyy:        parts[0],
    mmdd:        parts[1],
    hhmmss:      parts[2],
    lumins:      parseFloat(parts[3]),
    coordinates: [parseFloat(parts[4]), parseFloat(parts[5])]
  };
  const existing = JSON.parse(localStorage.getItem(STORAGE_KEY) || '[]');
  existing.push(entry);
  localStorage.setItem(STORAGE_KEY, JSON.stringify(existing));
}
